## 🧑‍💻 About Me

I'm a **full-stack engineer** focused on **shipping real, revenue-generating software** — not toys. My work centers around three themes:

- 🏢 **Multi-tenant SaaS platforms** — auth, billing, isolation, the unsexy plumbing that matters
- 💳 **Fintech & exchange infrastructure** — payments, KYC, real-time trading, multi-provider integrations
- 🤖 **AI-augmented operations** — turning LLMs into reliable workflow components, not chatbots

I design systems end-to-end: data model → API → frontend → deploy. I care about uptime, security, and shipping fast without leaving a mess.
